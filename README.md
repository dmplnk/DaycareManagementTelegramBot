# 🌈 Daycare Management Telegram Bot
Telegram-бот для автоматизации взаимодействия родителей и сотрудников детского сада.

Бот предоставляет родителям удобный способ получать информацию о ребёнке, просматривать расписание занятий и связываться с воспитателем без необходимости звонков или личного обращения

Проект разработан в рамках курсового проекта "Разработка базы данных для работы групп детского сада"

---

## ✨ Возможности

### 🔐 Авторизация
- вход по номеру телефона;
- проверка пользователя по базе данных;
- поддержка нескольких детей у одного родителя.
### 👨‍👩‍👧 Работа с ребёнком
- выбор ребёнка;
- просмотр группы;
- получение расписания занятий;
- просмотр контактов воспитателя и администрации.
### 💬 Взаимодействие с воспитателем
- уведомление «Стоим у двери»;
- запрос «Одеть ребёнка»;
- отправка сообщений воспитателю;
- уведомление о пропуске занятий;
- выбор даты отсутствия ребёнка.

## 🚀 Запуск проекта
### Установка зависимостей
pip install -r requirements.txt
### Настройка переменных окружения
Создайте файл .env:
```
DB_USER=
DB_HOST=
DB_NAME=
DB_PASSWORD=

BOT_TOKEN=
```
### Запуск
python bot.py
